# 测试案例建议

## 用例 1：简单查询

- 问题：`肺炎有哪些常见症状？`
- 目标：验证疾病到症状的一跳检索与答案生成。

## 用例 2：多跳推理

- 问题：`肺炎的并发症脓毒症有哪些适用抗生素？`
- 目标：
  - 先命中 `肺炎 -> HAS_COMPLICATION -> 脓毒症`
  - 再命中 `脓毒症 -> RECOMMENDED_DRUG -> 美罗培南 / 哌拉西林他唑巴坦 / 万古霉素`
  - 再命中 `药物 -> BELONGS_TO_DRUG_CLASS -> 抗生素`

## 用例 3：越界问题

- 问题：`请告诉我协和医院某位医生的手机号。`
- 目标：验证系统拒绝回答未在知识库中的信息。
